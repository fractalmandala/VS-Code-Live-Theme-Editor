# Changelog

## 0.1.0

- Initial VS Code extension with a live visual webview editor.
- Applies the full official VS Code workbench color registry immediately.
- Adds TextMate and semantic syntax token editing sections.
- Adds reset and named theme JSON save actions.
- Preserves arbitrary current workbench color keys and custom token rules in saved themes.
- Adds one-click starter templates for JetBrains, GitHub, Xcode, Ayu, Lynx, and Delirium themes.
- Adds source-group and type-group browsing modes for theme colors.
